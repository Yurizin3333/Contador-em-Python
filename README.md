# 🔢 Contador em Python

Projeto simples desenvolvido em Python com o objetivo de praticar **lógica de programação**, **estruturas de repetição** e **controle de fluxo**.

---

## 📌 Descrição

Este projeto implementa uma função chamada `contador`, capaz de realizar contagens de forma **crescente** ou **decrescente**, com base nos parâmetros informados pelo usuário.

O programa inclui:

* Contagem automática de **1 até 10**
* Contagem automática de **10 até 0 (de 2 em 2)**
* Um **contador personalizado**, onde o usuário define:

  * Início
  * Fim
  * Passo

---

## ⚙️ Funcionalidades

* ✔️ Contagem crescente e decrescente
* ✔️ Ajuste automático para passo negativo ou igual a zero
* ✔️ Entrada de dados pelo usuário
* ✔️ Simulação de contagem com `sleep()` (efeito visual)

---

## 🧠 Conceitos aplicados

* Estruturas condicionais (`if`, `elif`, `else`)
* Estruturas de repetição (`while`)
* Funções em Python
* Tratamento básico de entradas
* Controle de fluxo
* Manipulação de variáveis

## ▶️ Como executar

1. Certifique-se de ter o Python instalado (versão 3.x)
2. Clone este repositório:

```bash
git clone https://github.com/Yurizin3333/Contador-em-Python
```

3. Acesse a pasta do projeto:

```bash
cd Contador-em-Python
```

4. Execute o arquivo:

```bash
python contador.py
```

---

## 💻 Exemplo de uso

```bash
Contagem de 1 até 10 pulando 1 casa...
1 -> 2 -> 3 -> ... -> 10 -> FIM

Contagem de 10 até 0 diminuindo 2 casas...
10 -> 8 -> 6 -> ... -> 0 -> FIM

Contador personalizado...
Início: 5
Fim: 20
Passo: 3

5 -> 8 -> 11 -> 14 -> 17 -> 20 -> FIM
```
